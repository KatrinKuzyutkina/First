# Игра "Битва Кораблей"
**Битва Кораблей** - это консольная игра, в которой два игрока соревнуются друг с другом, стреляя по кораблям на игровом поле. Игра реализована на языке Python.
Игровое поле представляет собой сетку 10x10 клеток, где каждая клетка имеет свои координаты.
Каждый игрок имеет несколько кораблей различного размера, которые располагаются на игровом поле.
*Игра реализована на языке Python.*
## Цель
Пользователю необходимо потопить все корабли противника, попадая по их координатам.
## Правила игры
- Игроки по очереди делают ходы, указывая координаты клетки для стрельбы.
- Если игрок попадает по кораблю противника, то он получает дополнительный ход.
- Когда все клетки одного из кораблей противника подбиты, корабль считается потопленным.
- Побеждает тот игрок, который первым потопит все корабли противника.
## Реализация
1. Расстановка кораблей на поле:
   - Генерируется случайное поле, представляющее игровое поле.
   - Игрок самостоятельно выбирает координаты для размещения кодаблей.
   - Проверяется, не пересекаются ли корабли и не выходят ли за пределы поля.
   - Если корабли пересекаются или выходят за пределы поля, то алгоритм повторяется.
2. Ход игрока:
   - Игрок делает ход, указывая координаты выстрела на поле противника.
   - Проверяется, попал ли игрок в корабль противника.
   - Если игрок попал, то корабль противника повреждается и игрок получает дополнительный ход.
   - Если игрок не попал, то отображается сообщение о промахе и ход переходит к противнику.
3. Проверка окончания игры:
   - Проверяется, остались ли еще живые корабли на поле.
   - Происходит перебор всех клеток поля и проверки состояния каждой клетки.
   - Определение окончания игры при потоплении всех кораблей.
## Функциональные возможности
- Игрок может разместить свои корабли на игровом поле перед началом игры. Пользователю доступны несколько типов кораблей различной длины.
- Игрок может выбирать координаты на поле противника, чтобы совершить выстрел и попытаться попасть по кораблям противника.
- После выстрела игроку сообщается, попал ли он по кораблю противника или промахнулся.
- В левом верхнем углу пользователь видит количество кораблей противника, которые ему удалось утопить.
- Игра автоматически проверяет, когда все корабли одного из игроков потоплены, и объявляет окончание игры.
